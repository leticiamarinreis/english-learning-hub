# 📚 English Learning Hub

> A complete and interactive platform to master English verb tenses — with explanations, examples, notes, and progress tracking.

🌐 **Live Demo:** [https://leticiamarinreis.github.io/english-learning-hub](https://seu-usuario.github.io/english-learning-hub)

---

## ✨ Features

- **12 Verb Tenses** — from Present Simple to Future Perfect, each with full explanation, structure formulas, examples and exercises
- **Vocabulary** — key words with definitions, part of speech and example sentences
- **Grammar** — essential grammar rules (articles, conditionals, passive voice, modal verbs and more)
- **Notes** — personal note editor with save, load and delete, stored in localStorage
- **Progress Tracking** — tracks which tenses you have studied, stored in localStorage
- **Dark Mode** — toggle between light and dark themes
- **Responsive** — works on mobile and desktop

---

- Affirmative, Negative and Interrogative structures

## 🛠️ Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, responsive grid, dark mode, animations
- **JavaScript (ES6+)** — component-based architecture, service pattern, client-side routing
- **localStorage** — persistent notes and progress data
- **Google Fonts** — Fraunces (display) + DM Sans (body) + DM Mono (code)

> Although built as a single HTML file, the code follows Angular-inspired architecture with components, services, models and a client-side router.

## 🚀 How to Run Locally

No installation or build step needed. Just open the file in your browser:

```bash
# Clone the repository
git clone https://github.com/leticiamarinreis/english-learning-hub.git

# Open in browser
cd english-learning-hub
open index.html
```

Or simply double-click `index.html` on your computer.

---

## 🌐 Deploy with GitHub Pages

1. Push `index.html` to your repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your site will be live at `https://leticiamarinreis.github.io/english-learning-hub`
